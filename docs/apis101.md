## APIS End Points

- > [/metadata](##Me) \`returns all data needed for querying timetables.\` 
- > [/timetable](##Or) \`returns timetable data.\`

### Prerequisites

- Get Your API Key. add api key as query parameter at the end of URL. 
```html
    url?apikey=<your_api_key>
```
